\doxysubsection{Detailed Description}
Klasa reprezentująca jednostki kawalerii. 

Klasa implementująca specyficzne zachowania jednostek kawalerii. Charakteryzuje się największą prędkością poruszania się oraz specjalnymi zdolnościami takimi jak szarża i okrążanie przeciwników.
