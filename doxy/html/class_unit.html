Detailed Description

Bazowa klasa abstrakcyjna dla wszystkich jednostek w grze.

Klasa definiująca podstawowe atrybuty i zachowania wspólne dla wszystkich typów jednostek. Zawiera mechanizmy kolizji, ataku i obrony.
