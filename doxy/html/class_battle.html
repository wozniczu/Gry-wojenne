Detailed Description

Klasa zarządzająca przebiegiem bitwy.

Klasa odpowiedzialna za zarządzanie przebiegiem bitwy, tworzenie i aktualizację jednostek obu drużyn, sprawdzanie warunków zakończenia bitwy oraz zarządzanie stanem areny.
