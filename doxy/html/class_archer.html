Detailed Description

Klasa reprezentująca jednostki łuczników.

Klasa implementująca specyficzne zachowania jednostek łuczników. Charakteryzuje się atakami dystansowymi poprzez wystrzeliwanie strzał, jest jednak słaba w walce wręcz.
