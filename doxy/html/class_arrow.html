Detailed Description

Klasa reprezentująca strzałę wystrzeloną przez łucznika.

Klasa odpowiedzialna za reprezentację strzały w grze. Przechowuje informacje o pozycji, prędkości i stanie aktywności strzały, odpowiada za jej ruch i interakcję z celami.
